## Data Source
- **Files**:
  - `head_attention_judge.csv`
  - `head_attention_fan.csv`
  - `token_importance_contrib_judge.csv`
  - `token_importance_contrib_fan.csv`

- **Columns**:

  - Token columns: ballroom_partner, celebrity_industry, etc.
  - Values: Mean attention weights [0,1]

## Plotting Logic
Side-by-side heatmaps:
- Left: Judge attention allocation (4 heads × M tokens)
- Right: Fan attention allocation (4 heads × M tokens)
- Color scale: YlGnBu colormap

## Key Metrics
- JS Divergence (Judge vs Fan): Measures distribution difference
- Attention Entropy: Higher = more uniform, Lower = more concentrated
