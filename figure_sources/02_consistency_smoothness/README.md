## Paper Reference

- **Section**: Model I: Fan Vote Inference
- **Caption**: Elimination consistency vs temporal smoothness diagnostics

## Data Source
- **File**: `q1_consistency_with_smoothness_by_season.csv`
- **Generated by**: `task1/scripts/run_consistency_smoothness.py`
- **Columns**:
  - `season`: Season number
  - `elim_consistency`: L_s - fraction of weeks with correct elimination prediction
  - `smooth_penalty`: P_s - normalized temporal volatility penalty
  - `final_score`: Score_s = L_s - λ*P_s

## Plotting Logic
3-row × multi-season heatmap showing:
- Row 1: L_s (elimination consistency)
- Row 2: P_s (smoothness penalty, higher = worse)
- Row 3: Score_s (final composite score)
